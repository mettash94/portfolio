#1 - Diabetes Prediction Lab Portal



The dynamic web application serves as a portal for lab technicians to input patient data and predict the risk of Type 2 diabetes in female patients using the Naive Bayes machine learning algorithm. The logic is implented in Django a python web framework, and the prediciton uses the trained model based on the PIMA Indian Diabetes Dataset from the UCI Machine Learning Repository.










#2 - GUI based Scheduling Desktop Application



The standalone scheduling full stack application offers CRUD functionality, connecting to a MySQL database for customers and appointment management. Dynamic reports are generated based on database entities, and notifications alert users to appointments within 15 minutes of login. Localization supports multiple languages via .properties files. The app ensures time zone accuracy with time classes for global time handling.










#3 - BridXE Phydigital Fashion Marketplace E-Commerce App


The ecommerce app front end was developed for a phydigital fashion startup using modular, resuable components in React JS. Functionality and interactivity to components was implemented using React Router, Hooks and Redux.
